# Publicação

1. Apague/substitua os arquivos da raiz do repositório pelos arquivos desta versão.
2. Mantenha o GitHub Pages publicado pela branch `main`, pasta `/root`.
3. O domínio principal abrirá `site/`.
4. Painel: `https://angelsesteticaanimal.com.br/painel/`
5. Cliente: `https://angelsesteticaanimal.com.br/cliente/`
6. Após publicar, atualize com Ctrl+F5.
